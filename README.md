### Install VUE CLI through terminal
```bash
$ npm i -g @vue/cli @vue/cli-service-global
```
### Inside VSCode
VSCode terminal (ws is the folder where you want the project to be)
```c
$ vue create ws 
```
Then choose vue 2

```c
$ cd ws // folder we are working on
$ npm run serve // to run
```
